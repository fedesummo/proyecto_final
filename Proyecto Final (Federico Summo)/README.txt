----------------------------------------------- ElectronVolt ------------------------------------------------

Este proyecto simula ser el sitio web de mi futuro emprendimiento llamado "ElectronVolt". Este emprendimiento
brindará soluciones tecnológicas a empresas y particulares.
El sitio está compuesto por 5 páginas web, todas ellas caracterizadas por un diseño minimalista que combina
únicamente dos colores: el negro y el blanco.
El objetivo de este diseño consiste en ser amigable con el cliente para no abrumarlo con demasiada información
o con una multitud de recursos gráficos.
Todas las páginas del sitio son responsive, adaptandose a los distintos tamaños de pantalla.
A continuación, se explicará brevemente cada una de las páginas que componen al sitio:

  /- SERVICIOS (index) -/
Esta es la página principal del proyecto en la cual se comunican los distintos servicios que ofrece
el emprendimiento.

  /- TECNOLOGÍAS -/
En esta página se exponen los logotipos de las tecnologías que se utilizarán para el diseño y programación de
los sitios y aplicaciones web.

  /- OFERTAS -/
En esta página se exponen los precios de algunos de los servicios que el emprendimiento brindará.

  /- NOSOTROS -/
En esta página se expone información acerca del emprendimiento: surgimiento, objetivo profesional,
público objetivo. servicios prestados y forma de trabajo.

  /- CONTACTO -/
En esta página se comparten las redes sociales del emprendimiento. Además, se integra un formulario
de contacto.
